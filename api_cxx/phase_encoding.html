phase_encoding

Array<cfloat, 2> compas::phase_encoding(const CompasContext &ctx, const Array<cfloat, 2> &echos, const TissueParameters &parameters, const CartesianTrajectory &trajectory)

Apply phase encoding to the given magnetization.

Parameters:

  • ctx – Compas context.

  • echos – The magnetization at echo times. Size: [nreadouts, nvoxels].

  • parameters – The tissue parameters.

  • trajectory – The Trajectory.

Returns:

The phase encoded magnetization at echo times.
